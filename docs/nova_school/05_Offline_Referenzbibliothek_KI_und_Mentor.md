# Offline Referenzbibliothek, KI und Mentor

Nova School trennt bewusst zwischen **Wissenszugriff** und **freiem Internetzugriff**. Schueler sollen hochwertige Referenzen nutzen koennen, ohne dass ihre Projekte unkontrolliert nach aussen kommunizieren.

## Offline Referenzbibliothek

Die Referenzbibliothek liefert lokale Kopien offizieller oder primaerer Dokumentationen aus. Damit koennen Schueler suchen, lesen und nachschlagen, ohne direkten Webzugriff zu benoetigen.

Aktuelle Bereiche:

- Python
- JavaScript
- C++
- Java
- Rust
- HTML und CSS
- Node.js und npm
- Web Frontend
- Nova School Produktdokumentation

## Warum das sicherheitsrelevant ist

- Clients brauchen keinen offenen Webbrowser-Zugriff fuer Sprachdokumentation.
- Die Schule kontrolliert, welche Quellen importiert und freigegeben sind.
- Inhalte bleiben auch dann verfuegbar, wenn das Schulnetz ohne Internet arbeitet.

## LM Studio als Codehilfe

Die LM-Studio-Anbindung ist ein serverseitig verwalteter Dienst.

- Benutzer fragen ueber die Nova-School-Oberflaeche an.
- Der Zugriff kann pro Gruppe oder pro Nutzer erlaubt oder entzogen werden.
- Modell, Base URL und Freigaben werden zentral verwaltet.

## Sokratischer Mentor

Der Mentor ist keine reine Antwortmaschine. Er soll didaktisch begleiten:

- Rueckfragen statt Sofortloesungen
- Hinweise auf Fehlerbilder, Laufzeitspuren und Logik
- Unterstuetzung bei Refactoring und Verstaendnisfragen

## Unterschied zwischen KI-Hilfe und Mentor

| Funktion | Ziel | Typische Nutzung |
| --- | --- | --- |
| LM Studio Codehilfe | schnelle technische Assistenz | Code erklaeren, API finden, Syntax klaeren |
| Sokratischer Mentor | Lernprozess begleiten | Fehler reflektieren, Debugging verstehen, Denkweg trainieren |

## Empfehlungen

- KI nie als Ersatz fuer Rechte- und Aufgabenstruktur sehen.
- LM Studio nur dort aktivieren, wo die Schule den Einsatz didaktisch will.
- Mentor-Verlaeufe fuer Lernstandsdiagnose nutzen, nicht nur fuer Fehlerbehebung.
- Referenzbibliothek immer zuerst fuellen, bevor Webzugriff in Betracht gezogen wird.
